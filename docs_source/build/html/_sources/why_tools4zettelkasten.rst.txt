Why tools4zettelkasten?
=======================

tools4zettelkasten provides tools to setup a digital Zettelkasten System based
on simple interlinked markdown files.

A digital zettelkasten can be setup with just some folders and markdown files
and a good markdown editor. IMHO special database or cloud based systems are
not necessary.

zettelkasten_tools provide some tools to make handling of "markdown zettel
files" easy:

- helps to keep all files with a consistent naming convention
- helps reorganizing the file hierachy
- show your zettel notes in a browser as html
- creates graphical representation of your Zettelkasten, so that you can
  visualize your stem of thoughts
